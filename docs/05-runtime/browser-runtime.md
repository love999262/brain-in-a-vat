# 浏览器运行时

## 目标

`browser-runtime` 负责把引擎放进浏览器环境里稳定运行。

## 核心职责

- WebGPU 能力探测
- 页面可见性处理
- Worker 协调
- Service Worker 协调
- 多标签页主实例选举
- 运行等级判定

## 生命周期关注点

- 页面首次加载
- 资源准备完成
- 页面隐藏与恢复
- 标签页切换
- 多标签页竞争

## 输出能力

运行时至少需要向引擎暴露：

- 当前能力档位
- 当前是否为主实例
- 当前页面可见性
- 当前资源降级建议

## 边界

运行时不负责：

- 角色行为
- 关系计算
- UI 样式
