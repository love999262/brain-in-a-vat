# 能力分级

## 目标

能力分级用于把设备能力映射成可执行的运行策略。

## 输入因素

- WebGPU 是否可用
- 模型可用性
- 资源加载状态
- 浏览器运行限制
- 设备提示信息

## 输出内容

- 模型档位
- 同时可视角色数
- 同时激活角色数
- 动画质量
- 后台自动互动强度
- 降级状态

## 建议档位

- `full`
- `balanced`
- `lite`
- `safe`

## 移动端预留

V1 只保留以下扩展口：

- `deviceProfileHint`
- 降级状态上报
- 布局模式扩展

## 边界

能力分级影响运行时策略，不影响 Pack 内容结构。
