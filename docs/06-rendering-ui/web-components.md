# Web Components

## 目标

V1 只公开一个组件：`<brain-vat-world>`。

这个组件是引擎的最小 UI 入口，不是第二套业务层。

## 支持的接入方式

- 传入现成 `engine instance`
- 传入 `initConfig`，由组件内部创建引擎

## 组件职责

- 订阅 ViewModel
- 渲染基础舞台和面板
- 捕获用户输入
- 把用户意图转为 Command

## 不做的事

- 不直接修改业务状态
- 不定义第二套协议
- 不暴露一批未经稳定的内部子组件

## 主题与隔离

- 使用 Shadow DOM 做样式隔离
- 通过主题变量做外观扩展
- 不承诺宿主可直接操作内部 DOM 结构
