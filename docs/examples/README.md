# 示例文件

本文档目录用于放置结构化示例，帮助新协作者和新 AI 会话快速理解当前协议与 Pack 结构。

## 目录说明

- `character-pack/`
  - 角色包示例
- `source-pack/`
  - 消息源包示例
- `world-pack/`
  - 世界包示例
- `protocols/`
  - 初始化协议、Command、Event、ViewModel 示例

当前示例默认跟随官方演示世界，角色与世界实例名称使用 `Niziiro Mao`、`Hiyori Momose`、`Jin Natori` 这一组演示配置。

## 使用方式

- 阅读 Pack 规范时，可对照这里的 JSON 示例理解字段结构
- 实现 schema、加载器和初始化协议时，优先以这些示例为 fixture 基础
- 示例更新后，应同步检查文档是否仍然一致
