# Live2D 渲染层

## 目标

渲染层只负责角色如何展示，不负责角色为什么这样展示。

## 职责

- 加载 Live2D 资源
- 建立渲染实例
- 根据渲染状态切换动作和表情
- 提供 fallback 静态图
- 处理视觉降级

## 输入

渲染层只接收渲染相关输入，例如：

- 当前角色渲染配置
- 当前情绪标签
- 当前动作或表情映射
- 当前聚焦状态
- 当前是否发言

## 输出

- 资源加载状态
- 渲染实例状态
- 可供 `ResourceViewModel` 汇总的状态信息

## 边界

渲染层不应：

- 读取完整角色记忆
- 读取完整关系图
- 自己决定角色内容
